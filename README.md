# Apple system fonts extract
Extract apple system fonts downloaded from https://developer.apple.com/fonts/

# How to run
## Packages required

* [dmg2img](https://bit.ly/dmg2img)
* [p7zip-full](https://bit.ly/p7zip-full)

## Commands
* Extract fonts: `make`
* Create zip file for all fonts: `make release`
* Remove zip file: `make rm_release`
* Clean up: `make clean`
* List font files: `make list`
