# Apple system fonts extract
Extract apple system fonts downloaded from https://developer.apple.com/fonts/

# How to run
## Packages required

* [dmg2img](https://bit.ly/dmg2img)
* [p7zip-full](https://bit.ly/p7zip-full)

## Commands
* Extract fonts and create zip: `make` or `make all`
* Remove zip and clean up: `make clean`
