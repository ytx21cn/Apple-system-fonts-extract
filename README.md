# Apple system fonts extract
Extract apple system fonts downloaded from https://developer.apple.com/fonts/

# How to run
## Packages required

* [dmg2img](https://bit.ly/dmg2img)
* [xar](https://bit.ly/archive-xar)
* [p7zip-full](https://bit.ly/p7zip-full)
* [libxml2-dev](https://bit.ly/libxml2-dev)
* [libssl1.0.0, libssl1.0-dev](https://packages.ubuntu.com/search?suite=all&arch=any&searchon=names&keywords=libssl)
* [zlib1g-dev](https://packages.ubuntu.com/search?keywords=zlib1g-dev&searchon=names)

## Commands
* Unpack: `python3 ALL.py`
* Clean up: `python3 clean_up.py`
